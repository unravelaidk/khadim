Khadim — это опенсорсная, локальная платформа агентной автоматизации. Вместо готовых RPA-блоков Khadim использует AI-агентов для написания и выполнения скриптов автоматизации.

## Быстрый старт

```bash
npm install -g @unravelai/khadim
khadim --prompt "объясни этот проект"
```
