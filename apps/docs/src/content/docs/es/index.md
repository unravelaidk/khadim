Khadim es una plataforma de automatización agéntica local, de código abierto. En lugar de bloques RPA preconstruidos, Khadim utiliza agentes de IA para escribir y ejecutar scripts de automatización.

## Inicio rápido

```bash
npm install -g @unravelai/khadim
khadim --prompt "explica este proyecto"
```
