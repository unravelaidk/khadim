The repository includes example plugins you can copy and adapt while building your own tooling.

## Recommended starting points

- `examples/plugins/ts-hello-world` for the smallest useful TypeScript plugin.
- `examples/plugins/obsidian-wiki` for a more integrated example.

Use examples to understand:

- manifest structure
- host calls
- tool naming
- packaging flow
