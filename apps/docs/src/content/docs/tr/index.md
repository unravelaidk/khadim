Khadim, açık kaynaklı, yerel öncelikli bir ajan otomasyon platformudur. Hazır RPA blokları yerine Khadim, otomasyon betikleri yazmak ve çalıştırmak için AI ajanlarını kullanır.

## Hızlı başlangıç

```bash
npm install -g @unravelai/khadim
khadim --prompt "bu projeyi açıkla"
```
