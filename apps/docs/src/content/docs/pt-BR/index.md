Khadim é uma plataforma de automação agêntica local, de código aberto. Em vez de blocos RPA pré-construídos, o Khadim usa agentes de IA para escrever e executar scripts de automação.

## Início rápido

```bash
npm install -g @unravelai/khadim
khadim --prompt "explique este projeto"
```
