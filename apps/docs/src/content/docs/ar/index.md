خادم هو منصة أتمتة مفتوحة المصدر تعمل محليًا. بدلاً من كتل RPA المعدة مسبقًا، يستخدم خادم وكلاء الذكاء الاصطناعي لكتابة وتنفيذ نصوص الأتمتة.

## البداية السريعة

```bash
npm install -g @unravelai/khadim
khadim --prompt "اشرح هذا المشروع"
```
