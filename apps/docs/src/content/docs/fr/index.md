Khadim est une plateforme d'automatisation agentique locale et open-source. Au lieu de blocs RPA préconstruits, Khadim utilise des agents IA pour écrire et exécuter des scripts d'automatisation.

## Démarrage rapide

```bash
npm install -g @unravelai/khadim
khadim --prompt "explique ce projet"
```
