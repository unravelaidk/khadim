Khadim เป็นแพลตฟอร์มการทำงานอัตโนมัติด้วยเอเจนต์แบบโอเพนซอร์สและโลคอล แทนที่จะใช้บล็อก RPA สำเร็จรูป Khadim ใช้เอเจนต์ AI เพื่อเขียนและรันสคริปต์การทำงานอัตโนมัติ

## เริ่มต้นอย่างรวดเร็ว

```bash
npm install -g @unravelai/khadim
khadim --prompt "อธิบายโปรเจกต์นี้"
```
