Khadim este o platformă de automatizare agentică open-source, local-first. În loc de blocuri RPA predefinite, Khadim folosește agenți AI pentru a scrie și executa scripturi de automatizare.

## Început rapid

```bash
npm install -g @unravelai/khadim
khadim --prompt "explică acest proiect"
```
