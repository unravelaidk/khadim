ख़ादिम एक ओपन-सोर्स, लोकल-फर्स्ट एजेंट ऑटोमेशन प्लेटफॉर्म है। पहले से बने RPA ब्लॉक के बजाय, ख़ादिम AI एजेंट का उपयोग करता है जो ऑटोमेशन स्क्रिप्ट लिखते और चलाते हैं।

## त्वरित शुरुआत

```bash
npm install -g @unravelai/khadim
khadim --prompt "इस प्रोजेक्ट को समझाएं"
```
