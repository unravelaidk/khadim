Khadim er en åpen kildekode, lokal-først agentautomatiseringsplattform. I stedet for forhåndsbygde RPA-blokker bruker Khadim AI-agenter til å skrive og kjøre automatiseringsskript.

## Rask start

```bash
npm install -g @unravelai/khadim
khadim --prompt "forklar dette prosjektet"
```
