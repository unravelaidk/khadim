Khadim is een open-source, lokaal-eerst agentautomatiseringsplatform. In plaats van vooraf gebouwde RPA-blokken gebruikt Khadim AI-agents om automatiseringsscripts te schrijven en uit te voeren.

## Snelle start

```bash
npm install -g @unravelai/khadim
khadim --prompt "leg dit project uit"
```
