Khadim je open-source, lokalna platforma za agentnu automatizaciju. Umjesto unaprijed izgrađenih RPA blokova, Khadim koristi AI agente da pišu i izvršavaju skripte automatizacije.

## Brzi početak

```bash
npm install -g @unravelai/khadim
khadim --prompt "objasni ovaj projekat"
```
