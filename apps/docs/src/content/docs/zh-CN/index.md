Khadim 是一个开源、本地优先的智能体自动化平台。Khadim 使用 AI 智能体编写和执行自动化脚本，而非预构建的 RPA 模块。

## 快速开始

```bash
npm install -g @unravelai/khadim
khadim --prompt "解释这个项目"
```
