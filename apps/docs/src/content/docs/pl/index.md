Khadim to open-source'owa, lokalna platforma automatyzacji agentowej. Zamiast gotowych bloków RPA, Khadim wykorzystuje agentów AI do pisania i wykonywania skryptów automatyzacji.

## Szybki start

```bash
npm install -g @unravelai/khadim
khadim --prompt "wyjaśnij ten projekt"
```
