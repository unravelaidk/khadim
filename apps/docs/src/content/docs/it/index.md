Khadim è una piattaforma di automazione agentica open-source e locale. Invece di blocchi RPA predefiniti, Khadim utilizza agenti AI per scrivere ed eseguire script di automazione.

## Inizio rapido

```bash
npm install -g @unravelai/khadim
khadim --prompt "spiega questo progetto"
```
