Khadim はオープンソース、ローカルファーストのエージェント自動化プラットフォームです。既製の RPA ブロックの代わりに、Khadim は AI エージェントを使用して自動化スクリプトを作成・実行します。

## クイックスタート

```bash
npm install -g @unravelai/khadim
khadim --prompt "このプロジェクトを説明してください"
```
