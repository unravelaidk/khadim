Khadim ist eine Open-Source, lokal-zuerst Agentenautomatisierungsplattform. Anstatt vorgefertigter RPA-Blöcke verwendet Khadim KI-Agenten, um Automatisierungsskripte zu schreiben und auszuführen.

## Schnellstart

```bash
npm install -g @unravelai/khadim
khadim --prompt "erkläre dieses Projekt"
```
