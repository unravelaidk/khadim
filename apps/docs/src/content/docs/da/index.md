Khadim er en open-source, lokal-først agent automatiseringsplatform. I stedet for færdigbyggede RPA-blokke bruger Khadim AI-agenter til at skrive og udføre automatiseringsscripts.

## Hurtig start

```bash
npm install -g @unravelai/khadim
khadim --prompt "forklar dette projekt"
```
