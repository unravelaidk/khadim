Khadim은 오픈소스, 로컬 우선 에이전트 자동화 플랫폼입니다. 미리 만들어진 RPA 블록 대신 Khadim은 AI 에이전트를 사용하여 자동화 스크립트를 작성하고 실행합니다.

## 빠른 시작

```bash
npm install -g @unravelai/khadim
khadim --prompt "이 프로젝트를 설명해주세요"
```
