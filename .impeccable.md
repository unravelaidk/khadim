## Design Context

### Users
Developers using Khadim as a desktop AI-agent workspace — pairing with agents, reviewing diffs, managing workspaces, working with GitHub. Used in deep-focus sessions, often for long stretches, in varied lighting (day + night).

### Brand Personality
**Innovative, bold, warm.** Confident without being loud. Opinionated without being cold. Should feel like a well-made tool, not a web app pretending to be one.

### Aesthetic Direction
**Neutral ink** — clean, achromatic, content-first. Think Linear, Vercel, Raycast. Surfaces are near-neutral with barely perceptible cool tint. No colored brand hue polluting the interface — the accent is monochrome (near-white in dark, near-black in light). Color appears only for semantic meaning: green for success, red for danger, gold for attention.

**Themes:** Both light and dark are first-class. Dark leans cool charcoal (not warm, not blue-black). Light leans clean near-white (not parchment, not clinical).

**Anti-references:**
- NOT a SaaS dashboard (no card grids of metrics, no sparklines, no "hero number + label" tiles)
- NOT a Cursor/VSCode clone (no blue accents, no neon syntax, no cramped chrome)
- NOT a generic dev tool (no monospace-everywhere, no terminal-green accents)
- NOT warm/amber/orange-tinted — the default theme is deliberately achromatic

### Design Principles
1. **Typography is the interface.** Hierarchy comes from type scale and weight, not boxes and borders. A confident display face + a refined body face.
2. **Neutral palette, semantic color only.** Surfaces and accents are achromatic. Color is reserved for state: green=success, red=danger, gold=attention. No brand hue in the default theme.
3. **Chrome recedes, content leads.** Borders are 1px and optional. Backgrounds do most separation work. No nested cards.
4. **Motion is purposeful.** Staggered reveals on mount, smooth exponential easing, no bounce. Reduced motion is respected.
5. **Density with breathing room.** Dev tools need information density — but rhythm and whitespace make it readable. Varied spacing, never monotonous padding.
